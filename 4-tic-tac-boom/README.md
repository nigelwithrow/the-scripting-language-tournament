## Instructions
+ `$ cd 4-tic-tac-boom`
+ `$ yarn`
+ `$ yarn tsc`
+ `$ sed -r -i "s/export ?\{ ?.* ?\}.*$//" solution.js` # quickjs doesn't support 'export'
+ Have system-wide Rust & Cargo installation
+ `$ cargo build --release`
+ `$ ./target/tic-tac-boom solution.js`
